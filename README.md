# Analise-salarial-mercado-de-dados-e-arbitragem-remota-
Pipeline completo de dados (ETL + NLP + Visualização) analisando salários e mercado de tecnologia no Brasil e no mundo (2021–2024), usando dados do State of Data Brazil e Stack Overflow Survey. Inclui insights sobre senioridade, regiões, trabalho remoto, gênero, educação e ROI de tecnologias, com dashboard em tema Dark Neon.
